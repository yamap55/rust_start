# rust start
## preparation
1. install docker
2. install vscode
3. install vscode extension "Remote - Containers"
4. open vscode
5. `ctrl + shift + p` -> `Remote-Containers: Open Folder in Container`
6. select this folder

## usage
### run
```
cargo run
```

### build
```
cargo build
```

### build check
```
cargo check # or cargo c
```

### release build
```
cargo build --release
```
